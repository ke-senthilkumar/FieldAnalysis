# FieldAnalysis
Radiation Field Film Analysis

In Radiotherapy, analysis of the Radiation Field size is important as it gives some important information like:
1. Field Size
2. Flatness
3. Symmetry

There are a few commercial software and Pylinac for Python available. This is a small attempt in .Net analysing.
